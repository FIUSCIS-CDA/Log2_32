#### MODULE NAME
       Log2_32

#### DEPENDENCIES
       Encoder_32

#### INPUTS
Name | Size(Bits)
-----|------------
A   |    32     

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     5     

#### DESCRIPTION
       Computes the log-based 2 of a 32-bit unsigned integer.
